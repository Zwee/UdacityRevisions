Images, Alt Text, & Whitespace (not Whitesnake!)



Images with alt text

<img> tag is used to insert an image. It contains only attributes, and does NOT have a closing tag.

Please note: <img> is a void tag.

Why you should get in the habit of using alt text:

  • Lets you know something is preventing your picture from rendering on the page
  • Alt text can describe what's there/what was supposed to be there
  • Helpful for the visually impaired

Whitespace is the the all parts of the code where there is no code.

Whitespace is all the blank screen around the words. With other applications, like typing up a term paper in MS Word, whitespace can make a big difference. It can break up paragraphs, force text to display on an additional page, or make the text uncomfortable to read. But in HTML, whitespace doesn't matter.(*)

It doesn't matter how many times you hit your 'space' or 'enter' keys: unless you tell the computer where the breaks are, it will show everything in one looooooooong line.

'Break' or <br> is a void tag (and an inline tag) that provides line breaks. It just says 'the line ends here'. (Or, to quote Capt. Jean-Luc Picard: "The line must be drawn here!
