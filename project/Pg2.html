Tags and Attributes



About Tags

Tags can be nested.
Just keep an eye on the ones that operate in pairs, and make sure to close them in order.

An example would be the block tag <p></p>.
This tag creates 'paragraph blocks', and must be closed everytime. There must be a </p> for every <p> you use. (They get lonely!)

Some tags are "void tags", in that they have no content of their own so don't need to be closed.

About HTML Attributes

  • Attributes provide additional info about the element
  • Attributes come in pairs:
    • name/value
    • think: surname/Roddenbery, hair/Brown, age/70, status/Deceased
  • Attributes are always mentioned in the opening tag
  • Attribute names go in the opening tag
  • Attribute values go inside the quotation marks
  • Some attribute tags do not need to be closed, like <img>

A few examples:

  • <a>, </a> = anchor tag, used for making links
  • <a href...>, </a> = href tag (think Hyperlink Reference) (the 'href' is not repeated in the closing tag)
  • <img> = image tag. This tag does not need to be closed. It is used to add images.
