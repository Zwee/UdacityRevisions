WWW & HTML: What Are They?



The World-Wide Web
  • Uses HTML, or HyperTextMarkupLanguage, the main protocol on the web
  • Is the world's largest computer network
  • Is a collection of HTML docs
  • You/Your computer/Client (e.g.:Google)-- sends HTTP Request to Server
  • The Internet-- passes along the request to Server
  • Server-- receives HTTP Request and looks for/finds the requested HTML doc
  • Server-- sends info/HTML doc back
  • The Internet-- passes the info along to Client
  • Client/You/Your computer-- receives the doc, which is interpreted/displayed on screen

HTML is made up of Elements
HTML Elements contain:

  • Text Content - What you're looking at)
  • HTML Markup(++) - What it looks like
  • Referencess to other docs (e.g. images, videos)
  • Links to other pages (Other things to look at)

(++)HTML Markup consists of:

  1. an opening tag <tag name>
  2. the tag contents
  3. a closing tag </tag name>
HTML elements are either: 1) inline or 2) block.

As the name might suggest, Block makes an 'invisible box' around whatever is inside the element.
This 'box' has dimensions which can be exploited. (There will be more on this later.)

An Inline element occupies only the space reserved by the tags that define the inline element.
